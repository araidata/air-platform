# Update Status

Update repository status after a meaningful change.

Review and update:

- `docs/ai-context/current-state.md`
- `docs/ai-context/implementation-status.md`
- Relevant file in `docs/todos/`
- `docs/ai-context/known-issues.md`
- `docs/ai-context/next-steps.md`

Keep updates factual and brief. Record what changed, what is still deferred, and what should happen next.
