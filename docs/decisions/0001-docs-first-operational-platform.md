# ADR 0001 - Documentation-First Operational Platform

## Status

Accepted.

## Context

The platform must support long-term AI-assisted development and governance-focused implementation. The product domain is sensitive: public-sector AI risk, evidence, civil rights, security, privacy, and deployment approval.

## Decision

Create a complete documentation foundation before implementation code. Treat documentation as persistent engineering memory for human developers and AI coding assistants.

## Consequences

- Future implementation has clear constraints.
- AI assistants can resume work without rediscovering core architecture.
- Product direction is less likely to drift into chatbot or scanner-only patterns.
- Documentation must be maintained as implementation evolves.
