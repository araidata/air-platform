# Future Roadmap

The roadmap is assessment-led. Add technical capability only when it improves testing, evidence, findings, review decisions, or executive reporting.

## Near Term

- Giskard adapter.
- Giskard evidence preservation and finding normalization.
- OpenControl-ready mapping for automated test findings.
- Risk heatmap improvements.

## Mid Term

- PyRIT adapter.
- Multi-turn adversarial evidence handling.
- Langfuse trace ingestion.
- PDF executive reports.
- OpenControl / Compliance Masonry export.

## Later

- RBAC foundation.
- Backup and restore automation.
- Production logging and monitoring.
- Security hardening.
- Optional notification hooks.
- Additional scanner adapters when a concrete assessment workflow requires them.

## Explicitly Not Early

- Kubernetes.
- Scanner microservices.
- Multi-tenant SaaS architecture.
- Large-scale stream processing.
- Autonomous approval decisions.
- Chatbot-first product surface.
